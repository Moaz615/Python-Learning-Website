19. Functions and Parameters

Beginner

Organizing Code with Functions

A function is a block of organized, reusable code that is used to perform a single, related action. Functions provide better modularity for your application and a high degree of code reusing.

You define a function using the def keyword, followed by the function name, parentheses () (which can contain parameters), and a colon :. The function body is indented.

Function Definition Syntax 
def function_name(parameter1, parameter2, ...):
    """Docstring: explains what the function does."""
    # Function body
    # ...
    # return value # Optional: return a value

📤 Expected Output: 

Hello, Alice!
Hello, Bob!
Sum: 7
Product: 12

Parameters and Arguments

Parameters are the names listed in the function definition. They act as placeholders for the values the function expects to receive. Arguments are the actual values passed to the function when it is called.

Example: Greeting with a Parameter

Parameters and Arguments Example 
def greet(name): # 'name' is a parameter
    """This function greets the person passed in as a parameter."""
    print(f"Hello, {name}!")

# Calling the function with arguments
greet("Alice") # "Alice" is an argument
greet("Bob")   # "Bob" is an argument

# Function with multiple parameters
def add_numbers(num1, num2): # num1, num2 are parameters
    sum_result = num1 + num2
    print(f"The sum of {num1} and {num2} is: {sum_result}")

add_numbers(10, 5) # 10 and 5 are arguments
add_numbers(3.5, 2.1)

📤 Expected Output: 

Hello, Alice!
Hello, Bob!
The sum of 10 and 5 is: 15
The sum of 3.5 and 2.1 is: 5.6

Default Parameters

You can give parameters default values. If an argument is not provided for a parameter with a default value, the default value will be used.

Example:

Default Parameters Example 
def say_hello(name="Guest"): # "Guest" is the default value for name
    print(f"Hello, {name}!")

say_hello("Charlie") # Uses provided argument
say_hello()          # Uses default argument "Guest"

📤 Expected Output: 

Hello, Charlie!
Hello, Guest!
