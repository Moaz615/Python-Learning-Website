5. Data Types in Python

Beginner

What are Data Types?

Every value in Python has a data type. This type tells Python what kind of data it is and what operations can be performed on it. Understanding data types is crucial for writing correct and efficient code.

Some common built-in data types include:

Common Data Types

int Integers

Whole numbers (positive, negative, or zero)

10, -5, 0, 1000

float Floating-Point

Numbers with decimal points

3.14, -0.5, 2.0

str Strings

Sequences of characters

"hello", 'Python', "123"

bool Booleans

True or False values

True, False

list Lists

Ordered, mutable collections

[1, 2, 3], ["a", "b"]

dict Dictionaries

Key-value pairs

{"name": "Alice"}

Checking Data Types

You can use the built-in type() function to find out the data type of any variable or value.

Data Type Examples 
my_integer = 100
my_float = 25.5
my_string = "Hello World"
my_boolean = True
my_list = [1, 2, 3]

print(type(my_integer))
print(type(my_float))
print(type(my_string))
print(type(my_boolean))
print(type(my_list))
