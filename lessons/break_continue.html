17. Break and Continue

Beginner

Controlling Loop Execution

Sometimes, you need more control over how your loops execute. Python provides two keywords, break and continue, to alter the normal flow of loops.

  • `break` statement: Immediately terminates the loop (both `for` and `while` loops) and transfers control to the statement immediately following the loop.
  • `continue` statement: Skips the rest of the current iteration of the loop and moves to the next iteration.

The `break` Statement

Use break when you want to exit a loop early, perhaps because you found what you were looking for or an error condition occurred.

Break Statement Example 
# Search for a specific number
numbers = [1, 5, 8, 12, 15, 20]
target = 12

print("Searching for", target)
for num in numbers:
    if num == target:
        print(f"Found {target}!")
        break # Exit the loop once target is found
    print(f"Checking {num}...")
print("Loop finished.")

The `continue` Statement

Use continue when you want to skip the current iteration of a loop and move to the next one, without exiting the entire loop.

Continue Statement Example 
# Print only odd numbers
print("Odd numbers from 1 to 10:")
for i in range(1, 11):
    if i % 2 == 0 # If number is even
        continue # Skip to the next iteration
    print(i)

üì§ Expected Output: 

Odd numbers from 1 to 10:
1
3
5
7
9
