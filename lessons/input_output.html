6. Basic Input and Output

Beginner

Interacting with Your Program

Input and Output (I/O) are fundamental to making programs interactive.

OUT print()

Used to display output to the console

print("Hello, World!")

IN input()

Used to get input from the user

name = input("Enter name: ")

The `print()` Function

The print() function is one of the most commonly used functions in Python. It allows you to display text, numbers, and variable values on the screen.

print() Examples 
print("Hello, Python!")
name = "World"
print("Hello,", name, "!") # Multiple arguments separated by spaces
print(f"The number is {10 + 5}.") # f-string for formatted output

The `input()` Function

The input() function pauses your program and waits for the user to type something and press Enter. Whatever the user types is returned as a string.

input() Examples 
# In Pyodide, input() requires the user to type into the editor's input box
# and then click the "Run Code" button again to submit.
# For simplicity in this interactive environment, we'll simulate input.

# To properly test input(), you'd typically run this in a local terminal.
# In this web editor, input() might not behave as expected for real-time interaction.
# We'll use a placeholder for now.

# name = input("Enter your name: ")
# print(f"Hello, {name}!")

# For this environment, consider pre-setting variables for demonstration:
user_name = "Learner"
print(f"Hello, {user_name}! Welcome to I/O.")

age_str = "25" # Simulating input
age = int(age_str) # Convert string to integer
print(f"You are {age} years old.")

Note: The input() function in Pyodide (this web editor) works differently than in a standard Python terminal. It expects all input to be provided upfront. For interactive input, you might need to type your input into the editor and then re-run the code. For demonstration purposes, we'll often simulate input by assigning values directly.

Type Conversion

Since input() always returns a string, you often need to convert it to other data types.

int()

Convert to integer

int("25") ‚Üí 25

float()

Convert to float

float("3.14") ‚Üí 3.14

str()

Convert to string

str(42) ‚Üí "42"

Try it Yourself!

Use print() to display different messages. Experiment with converting string input to numbers.
