37. Regular Expressions (re module)

Intermediate Level

üîçWhat are Regular Expressions?

Regular Expressions (regex) are powerful patterns used to match, search, and manipulate text. They provide a concise and flexible way to identify strings of interest, such as particular characters, words, or patterns of characters.

In Python, regular expressions are handled by the re module, which provides a set of functions and methods for working with regex patterns.

Common Use Cases:

  • ‚Ä¢ Validating email addresses, phone numbers, or other formatted data
  • ‚Ä¢ Extracting specific information from text (like dates, URLs, etc.)
  • ‚Ä¢ Finding and replacing text patterns
  • ‚Ä¢ Parsing structured data from unstructured text
  • ‚Ä¢ Input validation and sanitization

‚ö°Basic Regex Patterns

Regular expressions use special characters and syntax to define patterns. Here are some fundamental patterns:

Character Classes

  • [abc] - Any character a, b, or c
  • [a-z] - Any lowercase letter
  • [0-9] - Any digit
  • [^abc] - Any character except a, b, c

Quantifiers

  • * - 0 or more occurrences
  • + - 1 or more occurrences
  • ? - 0 or 1 occurrence
  • {n} - Exactly n occurrences

Special Characters

  • . - Any character except newline
  • \w - Word character [a-zA-Z0-9_]
  • \d - Digit [0-9]
  • \s - Whitespace character

Anchors

  • ^ - Start of string
  • $ - End of string
  • \b - Word boundary
  • \B - Non-word boundary

üí°

Example: Common Regex Patterns

regex_basics.py 
import re

# Sample text for testing
text = "Hello World! My email is john.doe@example.com and my phone is 555-123-4567."

print("Original text:")
print(text)
print("\n" + "="*50)

# 1. Finding all words (sequences of letters)
words = re.findall(r'\b[a-zA-Z]+\b', text)
print("Words found:", words)

# 2. Finding email addresses
email_pattern = r'\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Z|a-z]{2,}\b'
emails = re.findall(email_pattern, text)
print("Email addresses found:", emails)

# 3. Finding phone numbers (US format)
phone_pattern = r'\b\d{3}-\d{3}-\d{4}\b'
phones = re.findall(phone_pattern, text)
print("Phone numbers found:", phones)

# 4. Replacing text
censored_text = re.sub(phone_pattern, "[PHONE]", text)
print("Text with censored phones:")
print(censored_text)

# 5. Splitting text
sentences = re.split(r'[.!?]+', text)
print("\nSentences:")
for i, sentence in enumerate(sentences, 1):
    if sentence.strip():
        print(f"{i}. {sentence.strip()}")

üîßAdvanced Regex Features

Regular expressions support advanced features like groups, lookaheads, and flags that make them even more powerful:

Groups and Capturing

  • () - Capturing group
  • (?:) - Non-capturing group
  • (?P<name>) - Named group

Flags

  • re.IGNORECASE - Case insensitive
  • re.MULTILINE - ^ and $ match line boundaries
  • re.DOTALL - . matches newlines
advanced_regex.py 
import re

# Sample data
data = """
Name: John Doe, Age: 30, Email: john@example.com
Name: Jane Smith, Age: 25, Email: jane.smith@test.org
Name: Bob Wilson, Age: 35, Email: bob_wilson@company.co.uk
"""

print("Extracting information with named groups:")
pattern = r"Name: (?P<name>[\w\s]+), Age: (?P<age>\d+), Email: (?P<email>[\w.@-]+)"

for match in re.finditer(pattern, data):
    print(f"Name: {match.group('name').strip()}")
    print(f"Age: {match.group('age')}")
    print(f"Email: {match.group('email')}")
    print("-" * 30)

print("\nCase insensitive matching:")
text = "Python is AWESOME! python is great. PYTHON is powerful."
python_matches = re.findall(re.compile(r"python", re.IGNORECASE), text)
print(f"Found 'python' {len(python_matches)} times (case insensitive)")

print("\nLookahead example:")
password = "MyPassword123!"
# Check if password has at least 8 chars, uppercase, lowercase, digit, and special char
strong_password_pattern = r"^(?=.*[a-z])(?=.*[A-Z])(?=.*\d)(?=.*[@$!%*?&])[A-Za-z\d@$!%*?&]{8,}$"
is_strong = re.match(strong_password_pattern, password)
print(f"Password '{password}' is strong: {is_strong is not None}")
