14. Sets

Beginner

Unique and Unordered Collections

A set is an unordered collection of unique items. This means that a set cannot have duplicate elements. Sets are useful when you need to store a collection of items and ensure that each item appears only once. Sets are defined by enclosing items in curly braces {}, similar to dictionaries, but they only contain values, not key-value pairs.

You can also create an empty set using set(), as {} creates an empty dictionary.

Set Operations Example

Let's explore the basic operations you can perform on sets, including adding, removing, and set operations like union and intersection.

Set Operations Example 
fruits = {"apple", "banana", "cherry"}
print("Original set:", fruits)

# Adding an element
fruits.add("orange")
print("After adding orange:", fruits)

# Removing an element
fruits.remove("banana")
print("After removing banana:", fruits)

# Membership test
print("Is apple in set?", "apple" in fruits)

# Iterating
print("All elements:")
for fruit in fruits:
    print(fruit)

# Set operations
odds = {1, 3, 5, 7}
evens = {2, 4, 6, 8}
primes = {2, 3, 5, 7}
print("Union:", odds | evens)
print("Intersection:", odds & primes)
print("Difference:", odds - primes)

Common Set Methods

Add/Remove Basic Operations

  • add(item) - Add an element
  • remove(item) - Remove element (raises error if not found)
  • discard(item) - Remove element (no error if not found)
  • pop() - Remove and return arbitrary element
  • clear() - Remove all elements

Operations Set Operations

  • union(other) - All elements from both sets
  • intersection(other) - Common elements
  • difference(other) - Elements in first but not second
  • symmetric_difference(other) - Elements in either but not both
  • issubset(other) - Check if all elements are in other set
