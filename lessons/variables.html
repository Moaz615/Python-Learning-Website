4. Variables in Python

Beginner

What are Variables?

In Python, a variable is like a container or a named storage location in your computer's memory. You use variables to store data that your program needs to work with. Think of it like labeling a box with a name and putting something inside it. Later, you can refer to that box by its label to get what's inside.

Python is dynamically typed, which means you don't need to declare the type of a variable (like integer, string, etc.) explicitly. Python figures it out automatically based on the value you assign to it.

Assigning Values to Variables

To create a variable and store a value in it, you use the assignment operator, which is the equals sign (=).

Variable Assignment Examples 
# Assigning an integer value
age = 30

# Assigning a string value
name = "Alice"

# Assigning a floating-point value
price = 19.99

# You can also assign the result of an expression
sum_numbers = 10 + 5

print(age)
print(name)
print(price)
print(sum_numbers)
