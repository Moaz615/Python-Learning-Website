9. String Operations

Beginner

Working with Text

A string is a sequence of characters, used for storing text information. In Python, strings are immutable, meaning once created, their content cannot be changed. You can create strings by enclosing characters in single quotes ('hello'), double quotes ("world"), or triple quotes for multi-line strings ("""multi-line""").

String Operations Example

String Operations Example 
my_string = "Python Programming"
print("Original string:", my_string)

# Accessing characters (strings are 0-indexed)
print("First character:", my_string[0])
print("Last character:", my_string[-1])

# Slicing (getting a part of the string)
print("Slice [0:6]:", my_string[0:6]) # Characters from index 0 up to (but not including) 6
print("Slice [7:]:", my_string[7:])   # From index 7 to the end

# String length
print("Length of string:", len(my_string))

# Concatenation (joining strings)
greeting = "Hello"
name = "Alice"
full_message = greeting + ", " + name + "!"
print(full_message)

# String methods
print("Uppercase:", my_string.upper())
print("Lowercase:", my_string.lower())
print("Replaced 'g' with 'X':", my_string.replace('g', 'X'))
print("Starts with 'Py':", my_string.startswith('Py'))
print("Ends with 'ing':", my_string.endswith('ing'))

Common String Methods

Basic Operations:

  • len(string) - Get length
  • string.upper() - Convert to uppercase
  • string.lower() - Convert to lowercase
  • string.strip() - Remove whitespace
  • string.split() - Split into list

Search & Replace:

  • string.find() - Find substring
  • string.replace() - Replace text
  • string.startswith() - Check prefix
  • string.endswith() - Check suffix
  • "text" in string - Check if contains

String Formatting

String Formatting Example 
name = "Alice"
age = 25
city = "New York"

# f-string formatting (Python 3.6+)
message = f"Hello, my name is {name}. I am {age} years old and live in {city}."
print(message)

# You can also use expressions inside f-strings
price = 19.99
quantity = 3
total = f"The total cost is ${price * quantity:.2f}"
print(total)

üì§ Expected Output: 

Hello, my name is Alice. I am 25 years old and live in New York.
The total cost is $59.97
