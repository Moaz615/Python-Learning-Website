Why Data Science Matters

Data Science is the art and science of extracting insights from data. Python is the most popular language for data science, thanks to its powerful libraries and active community.

Real-world applications:

  • Predicting trends in finance, healthcare, and business
  • Analyzing customer behavior and preferences
  • Building recommendation systems and AI models

Python's NumPy, Pandas, and scikit-learn are the core tools for data science workflows.

💡

Key Concepts: Data Science in Python

  • NumPy: Fast numerical operations and arrays.
  • Pandas: DataFrames for tabular data, cleaning, and analysis.
  • Data Cleaning: Handling missing values, outliers, and data types.
  • Feature Engineering: Creating new features for better models.
  • scikit-learn: Machine learning algorithms and model evaluation.
  • Matplotlib/Seaborn: Data visualization and exploration.

🔢NumPy Basics

NumPy is the foundational library for numerical computing in Python. It provides fast, memory-efficient arrays and a wide range of mathematical operations. In data science, NumPy is used for everything from basic statistics to advanced linear algebra.

💡

Example: Array Operations with NumPy

numpy_example.py 
import numpy as np

a = np.array([1, 2, 3])
b = np.array([4, 5, 6])
print("Sum:", a + b)
print("Dot product:", np.dot(a, b))

💡 Expected Output:

Sum: [5 7 9]
Dot product: 32

🧹Pandas Data Cleaning

Pandas makes it easy to load, clean, and analyze tabular data. Handling missing values is a crucial step in any data science workflow, as real-world data is often incomplete or messy. This example shows how to fill missing values with the mean of each column.

🧠

Example: Handling Missing Data with Pandas

pandas_cleaning.py 
import pandas as pd

data = {
    'A': [1, 2, None],
    'B': [4, None, 6]
}
df = pd.DataFrame(data)
print(df.fillna(df.mean()))

💡 Expected Output:

     A    B
0  1.0  4.0
1  2.0  5.0
2  1.5  6.0

🛠️Feature Engineering

Feature engineering is the process of creating new input features from your existing data to improve model performance. In this example, we calculate the Body Mass Index (BMI) from height and weight columns, adding a new, meaningful feature to the dataset.

✨

Example: Creating New Features

feature_engineering.py 
import pandas as pd

df = pd.DataFrame({
    'height_m': [1.70, 1.80, 1.65],
    'weight_kg': [65, 80, 55]
})
df['BMI'] = df['weight_kg'] / (df['height_m'] ** 2)
print(df)

💡 Expected Output:

   height_m  weight_kg        BMI
0      1.70         65  22.491349
1      1.80         80  24.691358
2      1.65         55  20.202020

🤖Model Training with scikit-learn

scikit-learn is the go-to library for machine learning in Python. This example demonstrates how to train a simple linear regression model, which learns the relationship between input features and a target variable. Model training is a core step in predictive data science workflows.

🏁

Example: Linear Regression

linear_regression.py 
from sklearn.linear_model import LinearRegression
import numpy as np

X = np.array([[1], [2], [3], [4]])
y = np.array([2, 4, 6, 8])

model = LinearRegression()
model.fit(X, y)
print("Predictions:", model.predict(X))

💡 Expected Output:

Predictions: [2. 4. 6. 8.]

📈Data Visualization with Matplotlib

Matplotlib is the most widely used library for creating static, interactive, and animated visualizations in Python. Visualizing your data helps you spot trends, outliers, and patterns that are not obvious from raw numbers. This example shows how to plot a simple line chart.

🎨

Example: Plotting a Line Chart

matplotlib_example.py 
import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [2, 4, 6, 8]

plt.plot(x, y)
plt.xlabel('X values')
plt.ylabel('Y values')
plt.title('Simple Line Chart')
plt.show()

💡 Expected Output:

A line chart window appears with the plotted data.

📝Data Analysis Example

This example demonstrates a typical data analysis workflow: creating a DataFrame, displaying it, and calculating a summary statistic (the average salary). These are the building blocks of most data science projects.

🔑

Example: Analyzing a CSV with Pandas

pandas_example.py 
import pandas as pd

data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'David'],
    'Age': [25, 30, 35, 40],
    'Salary': [70000, 80000, 120000, 90000]
}
df = pd.DataFrame(data)
print(df)
print("\nAverage salary:", df['Salary'].mean())

💡 Expected Output:

      Name  Age  Salary
0    Alice   25   70000
1      Bob   30   80000
2  Charlie   35  120000
3    David   40   90000

Average salary: 90000.0

✅ Best Practices for Data Science

  • Always inspect and clean your data before analysis.
  • Visualize data to spot trends and outliers.
  • Split data into training and test sets for machine learning.
  • Document your workflow and code for reproducibility.
  • Use version control for data and code.

⚠️ Common Pitfalls

  • Ignoring missing or inconsistent data.
  • Overfitting models to training data.
  • Not validating results with test data.
  • Misinterpreting statistical significance.
  • Failing to document data sources and changes.
