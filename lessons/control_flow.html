10. Conditional Statements (if, elif, else)

Beginner

Making Decisions with If/Else

In programming, conditional statements allow your program to make decisions and execute different blocks of code based on whether certain conditions are true or false. The primary conditional statements in Python are if, elif (short for "else if"), and else.

Basic Structure 
if condition_1:
    # Code to execute if condition_1 is True
elif condition_2:
    # Code to execute if condition_1 is False AND condition_2 is True
else:
    # Code to execute if all above conditions are False

📤 Expected Output: 

(Depends on the conditions and code inside each block)

The `if` Statement

The if statement is the most basic conditional statement. It executes a block of code only if its condition is True.

if Statement Example 
temperature = 28

if temperature > 25:
    print("It's a hot day!")
print("Done checking temperature.") # This line always runs

📤 Expected Output: 

It's a hot day!
Done checking temperature.

The `if-else` Statement

The else statement provides an alternative block of code to execute if the if condition is False.

if-else Statement Example 
age = 20

if age >= 18:
    print("You are an adult.")
else:
    print("You are a minor.")

📤 Expected Output: 

You are an adult.
You are a teenager.
You are a child.

The `if-elif-else` Statement

The elif (else if) statement allows you to check multiple conditions sequentially. Python checks conditions from top to bottom, and executes the first block whose condition is True, then skips the rest.

if-elif-else Statement Example 
score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
elif score >= 70:
    print("Grade: C")
else:
    print("Grade: F")

# Try changing the 'score' variable to see different outputs!

📤 Expected Output: 

Grade: B

Best Practices

✅ Do's:

  • • Use clear, descriptive condition names
  • • Keep conditions simple and readable
  • • Use proper indentation (4 spaces)
  • • Order conditions from most specific to least specific

❌ Don'ts:

  • • Don't nest too many if statements
  • • Don't forget the colon after conditions
  • • Don't mix tabs and spaces
  • • Don't use == for boolean checks
