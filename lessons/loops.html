16. Loops (For/While)

Beginner

Repeating Actions with Loops

Loops are used to execute a block of code repeatedly. This is incredibly useful when you need to perform the same action multiple times without writing the same code over and over. Python has two main types of loops: for loops and while loops.

  • For Loop: Used for iterating over a sequence (like a list, tuple, string, or range). It executes a block of code for each item in the sequence.
  • While Loop: Used to execute a block of code as long as a certain condition is true. It keeps repeating until the condition becomes false.

For Loop Example

For Loop Example 
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

print("\nCounting with range:")
for i in range(5): # range(5) generates numbers from 0 to 4
    print(i)

While Loop Example

While Loop Example 
count = 0
while count < 3:
    print("Count is:", count)
    count += 1 # Increment count to eventually make the condition false

üì§ Expected Output: 

Count is: 0
Count is: 1
Count is: 2
