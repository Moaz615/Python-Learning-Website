Key-Value Pairs with Dictionaries

A dictionary in Python is an unordered collection of data values, used to store data in a map-like fashion. Unlike lists, which are indexed by a range of numbers, dictionaries are indexed by keys. Each key is associated with a value, forming a key-value pair.

Dictionaries are mutable (changeable) and are defined by enclosing key-value pairs in curly braces {}, with each pair separated by a comma, and the key and value separated by a colon :.

Dictionary Operations Example

Let's explore the basic operations you can perform on dictionaries, including creating, accessing, modifying, and iterating through key-value pairs.

Dictionary Operations Example 
student = {
    "name": "Alice",
    "age": 20,
    "major": "Computer Science",
    "gpa": 3.8
}
print("Original dictionary:", student)

# Accessing values by key
print("Student's name:", student["name"])
print("Student's age:", student.get("age")) # Another way to get value

# Adding a new key-value pair
student["university"] = "Tech University"
print("After adding university:", student)

# Modifying a value
student["age"] = 21
print("After updating age:", student)

# Removing a key-value pair
del student["gpa"]
print("After deleting GPA:", student)

# Iterating through keys
print("\nKeys in dictionary:")
for key in student:
    print(key)

# Iterating through values
print("\nValues in dictionary:")
for value in student.values():
    print(value)

# Iterating through key-value pairs
print("\nItems in dictionary:")
for key, value in student.items():
    print(f"{key}: {value}")

Common Dictionary Methods

Access Getting Values

  • dict[key] - Get value by key
  • dict.get(key) - Get value safely
  • dict.get(key, default) - Get with default
  • dict.keys() - Get all keys
  • dict.values() - Get all values

Modify Modifying Dictionary

  • dict[key] = value - Add/update item
  • del dict[key] - Remove item
  • dict.pop(key) - Remove and return
  • dict.clear() - Remove all items
  • dict.update(other) - Merge dictionaries
