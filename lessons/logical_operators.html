11. Logical and Comparison Operators

Beginner

Combining Conditions and Making Comparisons

In Python, operators are special symbols that perform operations on values and variables. We've already looked at arithmetic operators. Now, let's dive into **Comparison Operators** (which you briefly saw in the Arithmetic & Comparison Operators lesson) and **Logical Operators**, which are crucial for making decisions in your code.

Comparison Operators (Revisit)

These operators compare two values and always return a Boolean result (`True` or `False`). They are the foundation for conditional statements.

Basic Comparisons:

  • == - Equal to
  • != - Not equal to
  • > - Greater than
  • < - Less than
  • >= - Greater than or equal to
  • <= - Less than or equal to

String Comparisons:

  • Strings are compared lexicographically
  • Case-sensitive by default
  • Use .lower() for case-insensitive
  • Numbers come before letters
Comparison Example 
num1 = 15
num2 = 10
string1 = "hello"
string2 = "Hello"

print(f"{num1} == {num2}: {num1 == num2}")
print(f"{num1} != {num2}: {num1 != num2}")
print(f"{num1} > {num2}: {num1 > num2}")
print(f"{string1} == {string2}: {string1 == string2}") # Case sensitive!
print(f"{string1.lower()} == {string2.lower()}: {string1.lower() == string2.lower()}")

📤 Expected Output: 

15 == 10: False
15 != 10: True
15 > 10: True
hello == Hello: False
hello == hello: True

Logical Operators

Logical operators are used to combine conditional statements (expressions that evaluate to `True` or `False`) and return a Boolean result. They are essential for building more complex decision-making logic in your programs.

AND Operator:

Returns True if BOTH conditions are true.

True and True = True
True and False = False
False and False = False

OR Operator:

Returns True if AT LEAST ONE condition is true.

True or True = True
True or False = True
False or False = False

NOT Operator:

Reverses the boolean result.

not True = False
not False = True

Examples of Logical Operators

Real-World Examples:

Logical Operators Example 
# --- AND operator ---
age = 25
has_license = True

if age >= 18 and has_license:
    print("You can drive.")
else:
    print("You cannot drive.")

# --- OR operator ---
is_weekend = True
has_holiday = False

if is_weekend or has_holiday:
    print("It's time to relax!")
else:
    print("It's a workday.")

# --- NOT operator ---
is_raining = False
if not is_raining:
    print("It's not raining, go outside!")
else:
    print("It's raining, stay inside.")

📤 Expected Output: 

You can drive.
It's time to relax!
It's not raining, go outside!

Operator Precedence

Order of Operations:

High Priority:

  • • not
  • • and
  • • or

Use Parentheses:

When in doubt, use parentheses to make the order clear:

(a and b) or (c and d)
