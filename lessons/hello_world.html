3. Hello World Program

Beginner

The Classic "Hello World"

The "Hello World" program is a tradition in programming. It's the first program most people write when learning a new language. It's simple but teaches you the basics of how to write and run code.

Fun Fact: The first "Hello World" program was written in 1972 by Brian Kernighan for the B programming language.

Writing Your First Program

Let's write a simple Hello World program. This will teach you about the print() function and how to use strings.

Hello World Example 
# This is my first Hello World program
print("Hello, World!")

# You can print multiple lines
print("Welcome to Python!")
print("I'm learning to program!")

# You can also print numbers
print(42)

# And combine text with numbers
print("The answer is:", 42)

Understanding the Code

1 print() Function

The print() function displays text or values on the screen.

print("Hello, World!")

2 Strings

Text in quotes is called a string. You can use single or double quotes.

"Hello, World!"
'Python is fun!'

3 Comments

Lines starting with # are comments that explain your code.

# This is a comment

4 Multiple Values

You can print multiple values by separating them with commas.

print("Text:", 42)
