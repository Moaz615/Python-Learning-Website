20. Return Values

Beginner

Getting Results from Functions

In the "Functions and Parameters" lesson, you learned how to define and call functions. Often, a function needs to perform some calculation or operation and then give back a result to the part of the code that called it. This is done using the `return` statement.

When Python encounters a `return` statement, it immediately exits the function and sends the specified value back to the caller. If a function doesn't have a `return` statement, it implicitly returns `None`.

Example: Calculating Area

Return Value Example 
def calculate_rectangle_area(length, width):
    """Calculates the area of a rectangle."""
    area = length * width
    return area # The function sends back the 'area' value

# Call the function and store its return value in a variable
room_length = 10
room_width = 5
room_area = calculate_rectangle_area(room_length, room_width)
print(f"The area of the room is: {room_area} square units.")

# You can also use the return value directly
print(f"Area of a 7x3 rectangle: {calculate_rectangle_area(7, 3)}")

📤 Expected Output: 

The area of the room is: 50 square units.
Area of a 7x3 rectangle: 21

Returning Multiple Values

Python functions can effectively return multiple values by packaging them into a tuple.

Returning Multiple Values Example 
def get_min_max(numbers):
    """Returns the minimum and maximum values from a list."""
    if not numbers:
        return None, None # Return two None values if list is empty
    return min(numbers), max(numbers) # Returns a tuple (min_val, max_val)

data = [10, 4, 20, 1, 15]
minimum, maximum = get_min_max(data) # Unpack the returned tuple
print(f"Minimum: {minimum}, Maximum: {maximum}")

empty_data = []
min_val, max_val = get_min_max(empty_data)
print(f"For empty list: Min={min_val}, Max={max_val}")

📤 Expected Output: 

Minimum: 1, Maximum: 20
For empty list: Min=None, Max=None
