23. Built-in Functions

Intermediate Level

üîéWhat Are Built-in Functions?

Python provides a collection of built-in functions ‚Äî tools that are always ready for you to use without needing to install anything. These functions help you perform common tasks like displaying output, converting data types, doing math, or working with collections.

You've already used some of them, like: print() to show messages, input() to take user input, and len() to count items in a list or string.

‚è±Ô∏èWhy Use Built-in Functions?

Built-in functions save time and make your code shorter and easier to read. You don't need to "reinvent the wheel" ‚Äî Python gives you a powerful toolbox, and knowing what's inside it is the first step to writing smarter code.

üß∞Common Built-in Functions

Here are some built-in functions you'll use often:

  • ‚û§abs(x): Gets the absolute value of a number (ignores the minus sign).
  • ‚û§max(...): Finds the largest value in a list or between values.
  • ‚û§min(...): Finds the smallest value.
  • ‚û§sum(iterable): Adds all the numbers in a list or tuple.
  • ‚û§round(number, ndigits): Rounds a number to a certain number of decimal places.
  • ‚û§type(object): Tells you the data type of something (e.g., int, float, str).
  • ‚û§int(), float(), str(), bool(): Used to convert between data types.
  • ‚û§range(start, stop, step): Creates a sequence of numbers, great for loops.
  • ‚û§sorted(iterable): Returns a new sorted version of a list without changing the original.
  • ‚û§all() and any(): Check if all or any items in a list are true.

How They Work (Examples)

Built-in Functions Example 
# Absolute value
print(abs(-15))  # Output: 15

# Find max and min
scores = [10, 2, 8, 15, 5]
print(max(scores))  # Output: 15
print(min(scores))  # Output: 2

# Add all numbers
print(sum(scores))  # Output: 40

# Round to 2 decimal places
print(round(3.14159, 2))  # Output: 3.14

# Check data types
print(type("hello"))  # Output: 
print(type(123))      # Output: 

# Convert data types
print(int("123"))     # Output: 123
print(float("49.99")) # Output: 49.99

# Sort a list
print(sorted([3, 1, 4, 1, 5]))  # Output: [1, 1, 3, 4, 5]

# Use all() and any()
print(all([True, True, True]))  # Output: True
print(any([False, False, True]))  # Output: True
