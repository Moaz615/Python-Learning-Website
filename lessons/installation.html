Beginner

Running Python Locally

While this website allows you to run Python code directly in your browser (thanks to Pyodide!), for more serious development, you'll want to install Python and a good code editor or Integrated Development Environment (IDE) on your own computer.

How to Install Python

The official way to get Python is from its website.

  1. Go to the official Python website: python.org/downloads
  2. Download the latest stable version for your operating system (Windows, macOS, Linux).
  3. Run the installer. Important: On Windows, make sure to check the box that says "Add Python X.Y to PATH" during installation. This makes it easier to run Python from your command line.
  4. Open your terminal (Command Prompt on Windows, Terminal on macOS/Linux) and type python --version or python3 --version to verify the installation.

Choosing an IDE or Code Editor

An IDE (Integrated Development Environment) or a powerful code editor makes writing Python code much easier with features like syntax highlighting, auto-completion, debugging, and more.

First Steps with Your Local Setup

Once you have Python and an editor installed:

  1. Open your chosen editor.
  2. Create a new file and save it as hello.py.
  3. Type print("Hello from local Python!") into the file.
  4. Open your terminal, navigate to the directory where you saved hello.py, and run: python hello.py (or python3 hello.py).

Expected Output: You should see "Hello from local Python!" printed in your terminal.
