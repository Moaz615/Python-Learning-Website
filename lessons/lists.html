📦 Organized Collections with Lists

A list is a versatile data type in Python that allows you to store an ordered collection of items. Lists are mutable, meaning you can change their content (add, remove, or modify items) after they are created. Lists are defined by enclosing items in square brackets [], with items separated by commas.

Items in a list can be of different data types.

🔧 List Operations Example

List Operations Example 
my_list = ["apple", "banana", "cherry"]
print("Original list:", my_list)

# Accessing elements (lists are 0-indexed)
print("First item:", my_list[0])
print("Last item:", my_list[-1])

# Modifying an element
my_list[1] = "blueberry"
print("Modified list:", my_list)

# Adding elements
my_list.append("orange") # Adds to the end
my_list.insert(0, "grape") # Inserts at a specific index
print("After adding:", my_list)

# Removing elements
my_list.remove("apple") # Removes first occurrence of value
popped_item = my_list.pop() # Removes and returns last item
print("After removing 'apple' and popping last:", my_list)
print("Popped item:", popped_item)

# Length of the list
print("Length of list:", len(my_list))

📤 Expected Output: 

[1, 2, 3, 4, 5]
3
[1, 2, 3, 4, 5, 6]
[2, 3, 4, 5, 6]
5
